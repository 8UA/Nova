# Nova

Nova is a web browser made with Python powered by **PyQt5/Webengine**

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
pip install -r requirements.txt
```

## Usage
```bash
python3 src/main.py
```
